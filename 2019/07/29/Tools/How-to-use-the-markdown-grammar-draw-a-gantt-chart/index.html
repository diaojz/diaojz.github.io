使用markdown语法绘制甘特图

发表于 2019-07-29 更新于 2020-01-02 分类于 Tools
本文字数： 526 阅读时长 ≈ 1 分钟

最近在项目中遇到需要绘制甘特图，使用了Excel绘制后，感觉并不顺手，不符合程序员的思维，于是搜索一番之后发现了更优雅的方式。

甘特图原理基本是一条线条图，横轴表示时间，纵轴表示活动(项目)，线条表示在整个期间上计划和实际的活动完成情况。它直观地表明任务计划在什么时候进行，及实际进展与计划要求的对比。
